<ActionHeader name={"send_msg"} summary={"发送消息（按 message_type/群号 自动路由群聊或私聊）"} readOnly={false} category={"消息"} />

Call via `POST http://127.0.0.1:3000/send_msg` (JSON params as the body) or WebSocket.

## Parameters

| Param | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `message` | message | ✓ |  |  |
| `message_type` | string | – |  |  |
| `group_id` | uint | – |  |  |
| `user_id` | uint | – |  |  |
| `auto_escape` | bool | – | `false` |  |

## Returns (data)

{ message_id: number }

Returns the standard OneBot envelope; the `data` shape is not yet documented.

## Examples

<Tabs>
<Tab label="curl">

```bash
curl -X POST http://127.0.0.1:3000/send_msg \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"message":"你好","message_type":"","group_id":123456,"user_id":10001,"auto_escape":false}'
```

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"message":"你好","message_type":"","group_id":123456,"user_id":10001,"auto_escape":false}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/send_msg", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // if an access token is configured
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
