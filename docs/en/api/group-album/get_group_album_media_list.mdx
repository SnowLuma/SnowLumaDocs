Call via `POST http://127.0.0.1:3000/get_group_album_media_list` (JSON params as the body) or WebSocket.

## Parameters

| Param | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `group_id` | uint | ✓ |  | 群号 |
| `album_id` | string | ✓ |  |  |
| `attach_info` | string | – | `""` |  |

## Returns (data)

相册媒体列表及下一页分页游标：{mediaList, nextAttachInfo}。

Every response is wrapped in the standard envelope: `{ "status": "ok", "retcode": 0, "data": ... }`. The table below describes `data`.

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `mediaList` | object[] | ✓ | 相册媒体项列表（各项字段不固定） |
| `nextAttachInfo` | string | ✓ | 下一页分页游标（空串表示无更多） |

<details>

```json
{
  "type": "object",

    "mediaList": {
      "type": "array",

      }

    "nextAttachInfo": {
      "type": "string",
      "description": "下一页分页游标（空串表示无更多）"

    "nextAttachInfo"
  ]
}
```

package main

import (

	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"group_id":123456,"album_id":"","attach_info":""}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/get_group_album_media_list", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // if an access token is configured
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
