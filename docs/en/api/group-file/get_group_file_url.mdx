<ActionHeader name={"get_group_file_url"} summary={"获取群文件下载链接"} readOnly={true} category={"群文件"} />

Call via `POST http://127.0.0.1:3000/get_group_file_url` (JSON params as the body) or WebSocket.

## Parameters

| Param | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `group_id` | uint | ✓ |  | 群号 |
| `file_id` | string | ✓ |  |  |
| `busid` | raw | – |  |  |

## Returns (data)

群文件下载链接。

Every response is wrapped in the standard envelope: `{ "status": "ok", "retcode": 0, "data": ... }`. The table below describes `data`.

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `url` | string | ✓ | 文件下载直链 |

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"group_id":123456,"file_id":"","busid":""}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/get_group_file_url", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // if an access token is configured
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
