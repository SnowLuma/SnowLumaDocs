Call via `POST http://127.0.0.1:3000/upload_private_file` (JSON params as the body) or WebSocket.

## Parameters

| Param | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `user_id` | uint | ✓ |  |  |
| `file` | string | ✓ |  |  |
| `name` | string | – | `""` |  |
| `upload_file` | bool | – | `true` |  |

## Returns (data)

{ file_id: string }

Returns the standard OneBot envelope; the `data` shape is not yet documented.

## Examples

<Tabs>
<Tab label="curl">

```bash
curl -X POST http://127.0.0.1:3000/upload_private_file \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"user_id":10001,"file":"","name":"","upload_file":true}'
```

</Tab>
<Tab label="Python">

```python
import requests

resp = requests.post(

    json={
        "user_id": 10001,
        "file": "",
        "name": "",
        "upload_file": True
    },
    headers={"Authorization": "Bearer <access-token>"},  # if an access token is configured
)
print(resp.json())
```

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"user_id":10001,"file":"","name":"","upload_file":true}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/upload_private_file", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // if an access token is configured
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
