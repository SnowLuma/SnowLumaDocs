# API Reference

OneBot v11-compatible API — 171 actions. Each is called via HTTP POST or WebSocket with a JSON body; responses use the OneBot envelope. Generated from `catalog.json`, sharing its source with runtime validation. (Action descriptions are authored in Chinese.)

:::tip Using an AI assistant?
MCP-capable clients can discover and call this action directly — no hand-written HTTP. See [MCP](/en/mcp/).
:::
