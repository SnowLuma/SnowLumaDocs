<ActionHeader name={"fetch_custom_face"} summary={"获取自定义表情"} readOnly={true} category={"扩展"} />

Call via `POST http://127.0.0.1:3000/fetch_custom_face` (JSON params as the body) or WebSocket.

## Parameters

| Param | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `count` | int | – | `10` |  |
| `return_type` | string | – | `"url"` |  |

## Returns (data)

字符串数组：return_type=url 时为图片 URL，return_type=id 时为 emoji_id。

`data` is `string[]`.

<details>
<summary>Raw JSON Schema</summary>

```json
{
  "type": "array",
  "items": {
    "type": "string",
    "description": "图片 URL 或 emoji_id（取决于 return_type）"
  }
}
```

</details>

## Examples

```bash

  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"count":10,"return_type":"url"}'
```

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"count":10,"return_type":"url"}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/fetch_custom_face", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // if an access token is configured
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
