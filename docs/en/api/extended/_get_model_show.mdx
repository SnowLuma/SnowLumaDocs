<ActionHeader name={"_get_model_show"} summary={"获取机型展示（兼容 mock）"} readOnly={true} category={"扩展"} />

Call via `POST http://127.0.0.1:3000/_get_model_show` (JSON params as the body) or WebSocket.

## Parameters

| Param | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `model` | string | – | `""` |  |

## Returns (data)

数组，每项含 variants（回显请求的机型名与 need_pay 标记）。

Every response is wrapped in the standard envelope: `{ "status": "ok", "retcode": 0, "data": ... }`. The table below describes `data`.

`data` is an array; each element:

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `variants` | object | ✓ |  |

<details>
<summary>Raw JSON Schema</summary>

```json
{
  "type": "array",
  "items": {
    "type": "object",
    "properties": {
      "variants": {
        "type": "object",
        "properties": {
          "model_show": {
            "type": "string",
            "description": "机型展示名（回显请求的 model，缺省 snowluma）"
          },
          "need_pay": {
            "type": "boolean",
            "description": "是否需付费（恒 false）"
          }
        },
        "required": [
          "model_show",
          "need_pay"
        ]
      }
    },
    "required": [
      "variants"
    ]
  }
}
```

</details>

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"model":""}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/_get_model_show", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // if an access token is configured
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
