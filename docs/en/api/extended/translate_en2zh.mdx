Call via `POST http://127.0.0.1:3000/translate_en2zh` (JSON params as the body) or WebSocket.

## Parameters

| Param | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `words` | raw | – |  |  |

## Returns (data)

\{ words \}：与输入等长的中文译文字符串数组。

Every response is wrapped in the standard envelope: `{ "status": "ok", "retcode": 0, "data": ... }`. The table below describes `data`.

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `words` | string[] | ✓ | 译文数组 |

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"words":""}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/translate_en2zh", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // if an access token is configured
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
