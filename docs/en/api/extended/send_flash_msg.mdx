<ActionHeader name={"send_flash_msg"} summary={"发送闪传消息（私聊或群聊，引用 fileset_id 让对端下载）"} readOnly={false} category={"扩展"} />

Call via `POST http://127.0.0.1:3000/send_flash_msg` (JSON params as the body) or WebSocket.

## Parameters

| Param | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `fileset_id` | string | ✓ |  |  |
| `user_id` | uint | – |  |  |
| `group_id` | uint | – |  |  |

## Returns (data)

{ message_id }

Returns the standard OneBot envelope; the `data` shape is not yet documented.

## Examples

<Tabs>
<Tab label="curl">

```bash
curl -X POST http://127.0.0.1:3000/send_flash_msg \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"fileset_id":"","user_id":10001,"group_id":123456}'
```

</Tab>

```python
print(resp.json())
```

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"fileset_id":"","user_id":10001,"group_id":123456}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/send_flash_msg", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // if an access token is configured
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
