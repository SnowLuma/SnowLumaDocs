<ActionHeader name={"get_forward_msg"} summary={"获取合并转发消息（id 或 message_id）"} readOnly={true} category={"扩展"} />

Call via `POST http://127.0.0.1:3000/get_forward_msg` (JSON params as the body) or WebSocket.

## Parameters

| Param | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `id` | string | – |  |  |

## Returns (data)

\{ messages \}：转发内的消息节点数组（每项为 OneBot 消息事件，内部字段不固定）。

Every response is wrapped in the standard envelope: `{ "status": "ok", "retcode": 0, "data": ... }`. The table below describes `data`.

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `messages` | object[] | ✓ | 转发消息节点数组 |

<details>
<summary>Raw JSON Schema</summary>

```json
{
  "type": "object",
  "properties": {
    "messages": {
      "type": "array",
      "description": "转发消息节点数组",
      "items": {
        "type": "object"
      }
    }
  },
  "required": [
    "messages"
  ]
}
```

</details>

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"id":""}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/get_forward_msg", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // if an access token is configured
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
