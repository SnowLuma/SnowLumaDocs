<ActionHeader name={"ocr_image"} summary={"OCR 图片（服务端，需图片 URL 或已缓存的图片 file_id）"} readOnly={true} category={"扩展"} aliases={[".ocr_image"]} />

Call via `POST http://127.0.0.1:3000/ocr_image` (JSON params as the body) or WebSocket.

## Parameters

| Param | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `image` | string | ✓ |  |  |

## Returns (data)

{ texts, language }：识别文本数组（含置信度与坐标）与识别语言。

Every response is wrapped in the standard envelope: `{ "status": "ok", "retcode": 0, "data": ... }`. The table below describes `data`.

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `texts` | object[] | ✓ | 识别出的文本块 |
| `language` | string | ✓ | 识别语言 |

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"image":""}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/ocr_image", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // if an access token is configured
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
