<ActionHeader name={"download_fileset"} summary={"解析闪传文件下载直链（不下载，由调用方实现下载）"} readOnly={false} category={"扩展"} />

Call via `POST http://127.0.0.1:3000/download_fileset` (JSON params as the body) or WebSocket.

## Parameters

| Param | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `fileset_id` | string | ✓ |  |  |
| `file_name` | string | – |  |  |
| `file_index` | number | – |  |  |

## Returns (data)

\{ url, file_name, file_size \}

Returns the standard OneBot envelope; the `data` shape is not yet documented.

## Examples

<Tabs>
<Tab label="curl">

```bash
curl -X POST http://127.0.0.1:3000/download_fileset \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"fileset_id":"","file_name":"","file_index":0}'
```

</Tab>
<Tab label="Python">

```python
import requests

resp = requests.post(
    "http://127.0.0.1:3000/download_fileset",
    json={
        "fileset_id": "",
        "file_name": "",
        "file_index": 0
    },
    headers={"Authorization": "Bearer <access-token>"},  # if an access token is configured
)
print(resp.json())
```

</Tab>

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"fileset_id":"","file_name":"","file_index":0}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/download_fileset", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // if an access token is configured
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
