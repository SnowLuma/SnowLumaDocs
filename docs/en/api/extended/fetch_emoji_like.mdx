<ActionHeader name={"fetch_emoji_like"} summary={"获取表情回应用户（NapCat 分页）"} readOnly={true} category={"扩展"} />

Call via `POST http://127.0.0.1:3000/fetch_emoji_like` (JSON params as the body) or WebSocket.

## Parameters

| Param | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `message_id` | messageId | ✓ |  |  |
| `emojiId` | string | ✓ |  |  |
| `count` | int | – | `10` |  |
| `cookie` | string | – | `""` |  |

## Returns (data)

分页的表情回应用户列表（NapCat 形状），含分页游标 cookie 与首/末页标记。

Every response is wrapped in the standard envelope: `{ "status": "ok", "retcode": 0, "data": ... }`. The table below describes `data`.

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `result` | integer | ✓ | 结果码（恒 0） |
| `errMsg` | string | ✓ | 错误信息（恒空串） |
| `emojiLikesList` | object[] | ✓ | 回应用户列表 |
| `cookie` | string | ✓ | 下一页游标（末页为空串） |
| `isLastPage` | boolean | ✓ | 是否末页 |
| `isFirstPage` | boolean | ✓ | 是否首页 |

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"message_id":100001,"emojiId":"","count":10,"cookie":""}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/fetch_emoji_like", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // if an access token is configured
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
