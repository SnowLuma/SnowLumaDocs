Call via `POST http://127.0.0.1:3000/create_collection` (JSON params as the body) or WebSocket.

## Parameters

This action takes no parameters.

## Returns (data)

Returns the standard OneBot envelope; the `data` shape is not yet documented.

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/create_collection", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // if an access token is configured
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
