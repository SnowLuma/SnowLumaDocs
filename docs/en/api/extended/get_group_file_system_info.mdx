<ActionHeader name={"get_group_file_system_info"} summary={"获取群文件系统信息"} readOnly={true} category={"扩展"} />

Call via `POST http://127.0.0.1:3000/get_group_file_system_info` (JSON params as the body) or WebSocket.

## Parameters

| Param | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `group_id` | uint | ✓ |  | 群号 |

## Returns (data)

\{ file_count, limit_count, used_space, total_space \}：群文件数量与容量信息（used_space 恒 0，total_space 为固定 10GiB 常量）。

Every response is wrapped in the standard envelope: `{ "status": "ok", "retcode": 0, "data": ... }`. The table below describes `data`.

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `file_count` | integer | ✓ | 当前文件数 |
| `limit_count` | integer | ✓ | 最大文件数 |
| `used_space` | integer | ✓ | 已用空间（占位，恒 0） |
| `total_space` | integer | ✓ | 总空间（占位，恒 10GiB） |

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"group_id":123456}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/get_group_file_system_info", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // if an access token is configured
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
