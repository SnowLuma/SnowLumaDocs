<ActionHeader name={"_del_group_notice"} summary={"删除群公告（fid 或 notice_id 二选一）"} readOnly={false} category={"扩展"} />

Call via `POST http://127.0.0.1:3000/_del_group_notice` (JSON params as the body) or WebSocket.

## Parameters

| Param | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `group_id` | uint | ✓ |  | 群号 |
| `fid` | string | – |  |  |
| `notice_id` | string | – |  |  |

## Returns (data)

Returns the standard OneBot envelope; the `data` shape is not yet documented.

## Examples

<Tabs>
<Tab label="curl">

```bash
curl -X POST http://127.0.0.1:3000/_del_group_notice \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"group_id":123456,"fid":"","notice_id":""}'
```

</Tab>
<Tab label="Python">

```python

        "notice_id": ""
    },
    headers={"Authorization": "Bearer <access-token>"},  # if an access token is configured
)
print(resp.json())
```

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"group_id":123456,"fid":"","notice_id":""}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/_del_group_notice", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // if an access token is configured
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
