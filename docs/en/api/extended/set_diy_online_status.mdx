Call via `POST http://127.0.0.1:3000/set_diy_online_status` (JSON params as the body) or WebSocket.

## Parameters

| Param | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `face_id` | uint | ✓ |  |  |
| `face_type` | int | – | `1` |  |
| `wording` | string | – | `""` |  |

## Returns (data)

Returns the standard OneBot envelope; the `data` shape is not yet documented.

## Examples

```bash

  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"face_id":0,"face_type":1,"wording":""}'
```

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"face_id":0,"face_type":1,"wording":""}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/set_diy_online_status", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // if an access token is configured
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
