<ActionHeader name={"get_qzone_msg_list"} summary={"获取 QQ 空间说说列表（默认机器人自己的空间）"} readOnly={true} category={"空间"} />

Call via `POST http://127.0.0.1:3000/get_qzone_msg_list` (JSON params as the body) or WebSocket.

## Parameters

| Param | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `target_uin` | uint | – |  | 目标 QQ 号，省略则取机器人自己 |
| `pos` | int | – | `0` | 起始偏移 |
| `num` | int | – | `20` | 本页数量 |

## Returns (data)

说说列表对象，含说说总数与本页说说数组。

Every response is wrapped in the standard envelope: `{ "status": "ok", "retcode": 0, "data": ... }`. The table below describes `data`.

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `total` | integer | ✓ | 账号说说总数（非本页数量） |
| `msglist` | object[] | ✓ | 本页说说数组 |

package main

import (

	"net/http"
)

func main() {

	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/get_qzone_msg_list", bytes.NewReader(body))

	req.Header.Set("Authorization", "Bearer <access-token>") // if an access token is configured

	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
