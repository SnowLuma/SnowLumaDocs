<ActionHeader name={"get_qun_album_list"} readOnly={true} category={"群相册"} />

通过 `POST http://127.0.0.1:3000/get_qun_album_list`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
| --- | --- | --- | --- | --- |
| `group_id` | uint | ✓ |  | 群号 |

## 返回 (data)

NapCat 风格的相册列表封套：\{album_list, attach_info, has_more\}（本实现 attach_info 恒为空串、has_more 恒为 false）。

所有响应都包裹在标准信封中：`{ "status": "ok", "retcode": 0, "data": ... }`。下表描述 `data` 字段。

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `album_list` | object[] | ✓ | 相册列表 |
| `attach_info` | string | ✓ | 分页游标（本 web 实现一次取满，恒为空串） |
| `has_more` | boolean | ✓ | 是否还有更多（本 web 实现恒为 false） |

```bash
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"group_id":123456}'
```

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"group_id":123456}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/get_qun_album_list", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
