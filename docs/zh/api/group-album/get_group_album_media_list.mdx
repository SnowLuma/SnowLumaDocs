<ActionHeader name={"get_group_album_media_list"} readOnly={true} category={"群相册"} />

通过 `POST http://127.0.0.1:3000/get_group_album_media_list`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
| --- | --- | --- | --- | --- |
| `group_id` | uint | ✓ |  | 群号 |
| `album_id` | string | ✓ |  |  |
| `attach_info` | string | – | `""` |  |

## 返回 (data)

相册媒体列表及下一页分页游标：\{mediaList, nextAttachInfo\}。

所有响应都包裹在标准信封中：`{ "status": "ok", "retcode": 0, "data": ... }`。下表描述 `data` 字段。

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `mediaList` | object[] | ✓ | 相册媒体项列表（各项字段不固定） |
| `nextAttachInfo` | string | ✓ | 下一页分页游标（空串表示无更多） |

<details>
<summary>原始 JSON Schema</summary>

```json
{
  "type": "object",
  "properties": {
    "mediaList": {
      "type": "array",
      "description": "相册媒体项列表（各项字段不固定）",
      "items": {
        "type": "object"
      }
    },
    "nextAttachInfo": {
      "type": "string",
      "description": "下一页分页游标（空串表示无更多）"
    }
  },
  "required": [
    "mediaList",
    "nextAttachInfo"
  ]
}
```

</details>

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"group_id":123456,"album_id":"","attach_info":""}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/get_group_album_media_list", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
