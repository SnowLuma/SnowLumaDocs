<ActionHeader name={"send_msg"} summary={"发送消息（按 message_type/群号 自动路由群聊或私聊）"} readOnly={false} category={"消息"} />

通过 `POST http://127.0.0.1:3000/send_msg`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
| --- | --- | --- | --- | --- |
| `message` | message | ✓ |  |  |
| `message_type` | string | – |  |  |
| `group_id` | uint | – |  |  |
| `user_id` | uint | – |  |  |
| `auto_escape` | bool | – | `false` |  |

## 返回 (data)

{ message_id: number }

该动作返回标准 OneBot 信封；`data` 结构待补充。

## 调用示例

<Tabs>
<Tab label="curl">

```bash
curl -X POST http://127.0.0.1:3000/send_msg \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"message":"你好","message_type":"","group_id":123456,"user_id":10001,"auto_escape":false}'
```

</Tab>

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"message":"你好","message_type":"","group_id":123456,"user_id":10001,"auto_escape":false}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/send_msg", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
