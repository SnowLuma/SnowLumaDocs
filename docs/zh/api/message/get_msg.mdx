<ActionHeader name={"get_msg"} summary={"获取消息"} readOnly={true} category={"消息"} />

通过 `POST http://127.0.0.1:3000/get_msg`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
| --- | --- | --- | --- | --- |
| `message_id` | messageId | ✓ |  |  |

## 返回 (data)

消息事件对象（首次收到时存储的副本，已去除 post_type/self_id、附带 real_id 字段并刷新图片 URL）。

该动作返回标准 OneBot 信封；`data` 结构待补充。

## 调用示例

<Tabs>
<Tab label="curl">

```bash
curl -X POST http://127.0.0.1:3000/get_msg \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"message_id":100001}'
```

</Tab>
<Tab label="Python">

```python
import requests

resp = requests.post(
    "http://127.0.0.1:3000/get_msg",
    json={
        "message_id": 100001
    },
    headers={"Authorization": "Bearer <access-token>"},  # 若已配置 access token
)
print(resp.json())
```

</Tab>

</Tab>
<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"message_id":100001}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/get_msg", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
