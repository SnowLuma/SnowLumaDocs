<ActionHeader name={"can_send_record"} readOnly={true} category={"信息"} />

通过 `POST http://127.0.0.1:3000/can_send_record`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

该动作无需参数。

## 返回 (data)

能力查询结果。

所有响应都包裹在标准信封中：`{ "status": "ok", "retcode": 0, "data": ... }`。下表描述 `data` 字段。

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `yes` | boolean | ✓ | 是否支持发送语音 |

<details>
<summary>原始 JSON Schema</summary>

```json
{
  "type": "object",
  "properties": {
    "yes": {
      "type": "boolean",
      "description": "是否支持发送语音"
    }
  },
  "required": [
    "yes"
  ]
}
```

</details>


<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/can_send_record", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
