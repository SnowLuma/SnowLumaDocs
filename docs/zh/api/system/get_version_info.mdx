通过 `POST http://127.0.0.1:3000/get_version_info`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

该动作无需参数。

## 返回 (data)

实现与协议版本信息。

所有响应都包裹在标准信封中：`{ "status": "ok", "retcode": 0, "data": ... }`。下表描述 `data` 字段。

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `app_name` | string | ✓ | 实现名称（SnowLuma） |
| `app_version` | string | ✓ | 实现版本 |
| `protocol_version` | string | ✓ | OneBot 协议版本 |

<details>
<summary>原始 JSON Schema</summary>

```json
{
  "type": "object",
  "properties": {
    "app_name": {
      "type": "string",
      "description": "实现名称（SnowLuma）"
    },
    "app_version": {
      "type": "string",
      "description": "实现版本"
    },
    "protocol_version": {
      "type": "string",
      "description": "OneBot 协议版本"
    }
  },
  "required": [
    "app_name",
    "app_version",
    "protocol_version"
  ]
}
```

</details>

```bash

  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{}'
```

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/get_version_info", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
