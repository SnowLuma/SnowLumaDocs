<ActionHeader name={"get_group_list"} summary={"获取群列表"} readOnly={true} category={"群信息"} />

通过 `POST http://127.0.0.1:3000/get_group_list`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
| --- | --- | --- | --- | --- |
| `no_cache` | bool | – | `false` |  |

## 返回 (data)

群信息对象数组。

所有响应都包裹在标准信封中：`{ "status": "ok", "retcode": 0, "data": ... }`。下表描述 `data` 字段。

`data` 为数组，每个元素：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `group_id` | integer | ✓ | 群号 |
| `group_name` | string | ✓ | 群名 |
| `member_count` | integer | ✓ | 当前成员数 |
| `max_member_count` | integer | ✓ | 成员上限 |

<details>


```bash

  -H 'Authorization: Bearer <access-token>' \
  -d '{"no_cache":false}'
```

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"no_cache":false}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/get_group_list", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
