<ActionHeader name={"get_group_root_files"} summary={"获取群根目录文件列表"} readOnly={true} category={"群文件"} />

通过 `POST http://127.0.0.1:3000/get_group_root_files`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
| --- | --- | --- | --- | --- |
| `group_id` | uint | ✓ |  | 群号 |

## 返回 (data)

群文件系统信息（文件与文件夹列表）。

所有响应都包裹在标准信封中：`{ "status": "ok", "retcode": 0, "data": ... }`。下表描述 `data` 字段。

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `files` | object[] | ✓ | 文件列表 |
| `folders` | object[] | ✓ | 文件夹列表 |

## 调用示例

<Tabs>
<Tab label="curl">

```bash
curl -X POST http://127.0.0.1:3000/get_group_root_files \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"group_id":123456}'
```

</Tab>
<Tab label="Python">

```python
import requests

resp = requests.post(
    "http://127.0.0.1:3000/get_group_root_files",
    json={
        "group_id": 123456
    },
    headers={"Authorization": "Bearer <access-token>"},  # 若已配置 access token
)
print(resp.json())
```

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"group_id":123456}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/get_group_root_files", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
