<ActionHeader name={"unlike_qzone"} summary={"取消对一条说说的点赞（QQ 空间；取消赞端点待真机核实）"} readOnly={false} category={"空间"} />

通过 `POST http://127.0.0.1:3000/unlike_qzone`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
| --- | --- | --- | --- | --- |
| `tid` | string | ✓ |  | 说说 tid |
| `target_uin` | uint | – |  | 说说所属 QQ 号，省略则为机器人自己 |
| `abstime` | int | – | `0` | 说说发表时间（unix 秒），传真实值更可靠 |

## 返回 (data)

该动作返回标准 OneBot 信封；`data` 结构待补充。

## 调用示例

<Tabs>
<Tab label="curl">

```bash
curl -X POST http://127.0.0.1:3000/unlike_qzone \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"tid":"","target_uin":0,"abstime":0}'
```

</Tab>

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"tid":"","target_uin":0,"abstime":0}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/unlike_qzone", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
