<ActionHeader name={"get_doubt_friends_add_request"} summary={"获取可疑好友申请"} readOnly={true} category={"扩展"} />

通过 `POST http://127.0.0.1:3000/get_doubt_friends_add_request`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
| --- | --- | --- | --- | --- |
| `count` | int | – | `50` |  |

## 返回 (data)

可疑好友申请数组，每项含 uid（作为处理用 flag）、昵称、来源、留言与申请时间。

所有响应都包裹在标准信封中：`{ "status": "ok", "retcode": 0, "data": ... }`。下表描述 `data` 字段。

`data` 为数组，每个元素：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `uid` | string | ✓ | 申请人 uid（回传作 set_doubt_friends_add_request 的 flag） |
| `nick` | string | ✓ | 申请人昵称 |
| `source` | string | ✓ | 申请来源 |
| `msg` | string | ✓ | 验证留言 |
| `reqTime` | integer | ✓ | 申请时间戳 |

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"count":50}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/get_doubt_friends_add_request", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
