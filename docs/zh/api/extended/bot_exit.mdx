<ActionHeader name={"bot_exit"} summary={"退出机器人"} readOnly={false} category={"扩展"} />

通过 `POST http://127.0.0.1:3000/bot_exit`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

该动作无需参数。

## 返回 (data)

该动作返回标准 OneBot 信封；`data` 结构待补充。

## 调用示例

<Tabs>
<Tab label="curl">

```bash
curl -X POST http://127.0.0.1:3000/bot_exit \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{}'
```

</Tab>
<Tab label="Python">

```python
import requests

resp = requests.post(
    "http://127.0.0.1:3000/bot_exit",
    json={},
    headers={"Authorization": "Bearer <access-token>"},  # 若已配置 access token
)
print(resp.json())
```

</Tab>
<Tab label="JavaScript">

```js
const resp = await fetch("http://127.0.0.1:3000/bot_exit", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "Authorization": "Bearer <access-token>", // 若已配置 access token
  },
  body: JSON.stringify({}),
});
console.log(await resp.json());
```

</Tab>
<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/bot_exit", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```

</Tab>
<Tab label="SnowLuma SDK">

```ts
import { SnowLumaHttpClient } from '@snowluma/sdk';

const bot = new SnowLumaHttpClient({
  baseUrl: 'http://127.0.0.1:3000/',
  accessToken: process.env.SNOWLUMA_TOKEN, // 若已配置 access token
});

const data = await bot.call('bot_exit', {});
console.log(data);
```
