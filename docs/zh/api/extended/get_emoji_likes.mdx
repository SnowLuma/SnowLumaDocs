<ActionHeader name={"get_emoji_likes"} summary={"获取表情回应用户"} readOnly={true} category={"扩展"} />

通过 `POST http://127.0.0.1:3000/get_emoji_likes`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
| --- | --- | --- | --- | --- |
| `message_id` | messageId | ✓ |  |  |
| `emoji_id` | string | ✓ |  |  |

## 返回 (data)

{ emoji_like_list }：回应该表情的用户列表（nick_name 恒为空串）。

所有响应都包裹在标准信封中：`{ "status": "ok", "retcode": 0, "data": ... }`。下表描述 `data` 字段。

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `emoji_like_list` | object[] | ✓ | 回应用户列表 |

<details>
<summary>原始 JSON Schema</summary>

```json
{
  "type": "object",
  "properties": {
    "emoji_like_list": {
      "type": "array",
      "description": "回应用户列表",
      "items": {
        "type": "object",
        "properties": {
          "user_id": {
            "type": "string",
            "description": "用户 QQ 号（字符串）"
          },
          "nick_name": {
            "type": "string",
            "description": "昵称（当前实现恒为空串）"
          }
        },
        "required": [
          "user_id",
          "nick_name"
        ]
      }
    }
  },
  "required": [
    "emoji_like_list"
  ]
}
```

</details>

<Tab label="curl">


<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"message_id":100001,"emoji_id":""}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/get_emoji_likes", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
