<ActionHeader name={"get_ai_characters"} summary={"获取 AI 语音角色"} readOnly={true} category={"扩展"} />

通过 `POST http://127.0.0.1:3000/get_ai_characters`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
| --- | --- | --- | --- | --- |
| `group_id` | uint | ✓ |  | 群号 |
| `chat_type` | int | – | `1` |  |

## 返回 (data)

按分类分组的 AI 语音角色列表，每组含分类名与角色（id、名称、试听 URL）。

所有响应都包裹在标准信封中：`{ "status": "ok", "retcode": 0, "data": ... }`。下表描述 `data` 字段。

`data` 为数组，每个元素：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `type` | string | ✓ | 角色分类名 |
| `characters` | object[] | ✓ | 该分类下的角色列表 |

```json
    "required": [
      "type",

    ]
  }
}
```

## 调用示例

<Tabs>
<Tab label="curl">

```bash
curl -X POST http://127.0.0.1:3000/get_ai_characters \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"group_id":123456,"chat_type":1}'
```

</Tab>

```python
    },
    headers={"Authorization": "Bearer <access-token>"},  # 若已配置 access token
```

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"group_id":123456,"chat_type":1}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/get_ai_characters", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
