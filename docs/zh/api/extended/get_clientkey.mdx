<ActionHeader name={"get_clientkey"} summary={"获取 clientkey"} readOnly={true} category={"扩展"} />

通过 `POST http://127.0.0.1:3000/get_clientkey`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

该动作无需参数。

## 返回 (data)

{ clientKey, expireTime, keyIndex }：clientkey 及其过期时间与索引。

所有响应都包裹在标准信封中：`{ "status": "ok", "retcode": 0, "data": ... }`。下表描述 `data` 字段。

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `clientKey` | string | ✓ | clientkey |
| `expireTime` | string | ✓ | 过期时间 |
| `keyIndex` | string | ✓ | key 索引 |

<details>
<summary>原始 JSON Schema</summary>

```json
{
  "type": "object",
  "properties": {
    "clientKey": {
      "type": "string",
      "description": "clientkey"
    },
    "expireTime": {
      "type": "string",
      "description": "过期时间"
    },
    "keyIndex": {
      "type": "string",
      "description": "key 索引"
    }
  },
  "required": [
    "clientKey",
    "expireTime",
    "keyIndex"
  ]
}
```

</details>

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/get_clientkey", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
