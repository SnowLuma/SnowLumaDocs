<ActionHeader name={"get_recent_contact"} summary={"获取最近会话（占位）"} readOnly={true} category={"扩展"} />

通过 `POST http://127.0.0.1:3000/get_recent_contact`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
| --- | --- | --- | --- | --- |
| `count` | int | – | `10` |  |

## 返回 (data)

占位实现，恒返回空数组。

`data` 为 `any[]`。

<details>
<summary>原始 JSON Schema</summary>

```json
{
  "type": "array",
  "description": "最近会话列表（占位，恒空）"
}
```

</details>

## 调用示例

<Tabs>
<Tab label="curl">

```bash
curl -X POST http://127.0.0.1:3000/get_recent_contact \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"count":10}'
```

</Tab>
<Tab label="Python">

```python
import requests

resp = requests.post(
    "http://127.0.0.1:3000/get_recent_contact",
    json={
        "count": 10
    },
    headers={"Authorization": "Bearer <access-token>"},  # 若已配置 access token
)
print(resp.json())
```

</Tab>

</Tab>
<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"count":10}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/get_recent_contact", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
