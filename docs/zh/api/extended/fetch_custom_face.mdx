<ActionHeader name={"fetch_custom_face"} summary={"获取自定义表情"} readOnly={true} category={"扩展"} />

通过 `POST http://127.0.0.1:3000/fetch_custom_face`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
| --- | --- | --- | --- | --- |
| `count` | int | – | `10` |  |
| `return_type` | string | – | `"url"` |  |

## 返回 (data)

字符串数组：return_type=url 时为图片 URL，return_type=id 时为 emoji_id。

`data` 为 `string[]`。

<details>
<summary>原始 JSON Schema</summary>

```json
{
  "type": "array",
  "items": {
    "type": "string",
    "description": "图片 URL 或 emoji_id（取决于 return_type）"
  }
}
```

</details>

## 调用示例

<Tabs>
<Tab label="curl">

```bash
curl -X POST http://127.0.0.1:3000/fetch_custom_face \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"count":10,"return_type":"url"}'
```

</Tab>
<Tab label="Python">

```python
import requests

resp = requests.post(
    "http://127.0.0.1:3000/fetch_custom_face",
    json={
        "count": 10,
        "return_type": "url"
    },
    headers={"Authorization": "Bearer <access-token>"},  # 若已配置 access token
)
print(resp.json())
```

</Tab>

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"count":10,"return_type":"url"}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/fetch_custom_face", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
