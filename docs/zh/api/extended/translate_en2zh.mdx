通过 `POST http://127.0.0.1:3000/translate_en2zh`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
| --- | --- | --- | --- | --- |
| `words` | raw | – |  |  |

## 返回 (data)

{ words }：与输入等长的中文译文字符串数组。

所有响应都包裹在标准信封中：`{ "status": "ok", "retcode": 0, "data": ... }`。下表描述 `data` 字段。

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `words` | string[] | ✓ | 译文数组 |

<details>
<summary>原始 JSON Schema</summary>

```json
{
  "type": "object",
  "properties": {
    "words": {
      "type": "array",
      "description": "译文数组",
      "items": {
        "type": "string"
      }
    }
  },
  "required": [
    "words"
  ]
}
```

</details>

## 调用示例

<Tabs>
<Tab label="curl">

```bash
curl -X POST http://127.0.0.1:3000/translate_en2zh \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"words":""}'
```

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"words":""}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/translate_en2zh", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
