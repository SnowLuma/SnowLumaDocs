<ActionHeader name={"set_self_longnick"} summary={"设置个性签名（longNick/long_nick，严格 string）"} readOnly={false} category={"扩展"} />

通过 `POST http://127.0.0.1:3000/set_self_longnick`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
| --- | --- | --- | --- | --- |
| `longNick` | raw | – |  |  |
| `long_nick` | raw | – |  |  |

## 返回 (data)

该动作返回标准 OneBot 信封；`data` 结构待补充。

## 调用示例

<Tabs>
<Tab label="curl">

```bash
curl -X POST http://127.0.0.1:3000/set_self_longnick \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"longNick":"","long_nick":""}'
```

</Tab>
<Tab label="Python">

```python
import requests

resp = requests.post(
    "http://127.0.0.1:3000/set_self_longnick",
    json={
        "longNick": "",
        "long_nick": ""
    },
    headers={"Authorization": "Bearer <access-token>"},  # 若已配置 access token
)
print(resp.json())
```

</Tab>
<Tab label="JavaScript">

</Tab>
<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"longNick":"","long_nick":""}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/set_self_longnick", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
