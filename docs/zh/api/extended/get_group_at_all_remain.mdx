<ActionHeader name={"get_group_at_all_remain"} summary={"获取群 @全体成员 剩余次数"} readOnly={true} category={"扩展"} />

通过 `POST http://127.0.0.1:3000/get_group_at_all_remain`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
| --- | --- | --- | --- | --- |
| `group_id` | uint | ✓ |  | 群号 |

## 返回 (data)

\{ can_at_all, remain_at_all_count_for_group, remain_at_all_count_for_uin \}：@全体可用性与剩余次数。

所有响应都包裹在标准信封中：`{ "status": "ok", "retcode": 0, "data": ... }`。下表描述 `data` 字段。

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `can_at_all` | boolean | ✓ | 当前是否可 @全体成员 |
| `remain_at_all_count_for_group` | integer | ✓ | 本群今日剩余 @全体次数 |
| `remain_at_all_count_for_uin` | integer | ✓ | 本账号今日剩余 @全体次数 |

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"group_id":123456}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/get_group_at_all_remain", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
