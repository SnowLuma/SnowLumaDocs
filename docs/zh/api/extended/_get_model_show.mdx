<ActionHeader name={"_get_model_show"} summary={"获取机型展示（兼容 mock）"} readOnly={true} category={"扩展"} />

通过 `POST http://127.0.0.1:3000/_get_model_show`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
| --- | --- | --- | --- | --- |
| `model` | string | – | `""` |  |

## 返回 (data)

数组，每项含 variants（回显请求的机型名与 need_pay 标记）。

所有响应都包裹在标准信封中：`{ "status": "ok", "retcode": 0, "data": ... }`。下表描述 `data` 字段。

`data` 为数组，每个元素：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `variants` | object | ✓ |  |

<details>
<summary>原始 JSON Schema</summary>

```json
{
  "type": "array",
  "items": {
    "type": "object",
    "properties": {
      "variants": {
        "type": "object",
        "properties": {
          "model_show": {
            "type": "string",
            "description": "机型展示名（回显请求的 model，缺省 snowluma）"
          },
          "need_pay": {
            "type": "boolean",
            "description": "是否需付费（恒 false）"
          }
        },
        "required": [
          "model_show",
          "need_pay"
        ]
      }
    },
    "required": [
      "variants"
    ]
  }
}
```

</details>

## 调用示例

<Tabs>
<Tab label="curl">

```bash
curl -X POST http://127.0.0.1:3000/_get_model_show \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"model":""}'
```

</Tab>


<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"model":""}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/_get_model_show", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
