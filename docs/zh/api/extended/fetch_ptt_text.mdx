<ActionHeader name={"fetch_ptt_text"} summary={"获取语音转文字结果"} readOnly={true} category={"扩展"} aliases={["get_ptt_text","get_record_text"]} />

通过 `POST http://127.0.0.1:3000/fetch_ptt_text`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
| --- | --- | --- | --- | --- |
| `message_id` | string | – | `""` |  |

## 返回 (data)

\{ text \}：语音识别出的文本。

所有响应都包裹在标准信封中：`{ "status": "ok", "retcode": 0, "data": ... }`。下表描述 `data` 字段。

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `text` | string | ✓ | 语音转写文本 |

<details>
<summary>原始 JSON Schema</summary>

```json
{
  "type": "object",
  "properties": {
    "text": {
      "type": "string",
      "description": "语音转写文本"
    }
  },
  "required": [
    "text"
  ]
}
```

</details>

## 调用示例

<Tabs>
<Tab label="curl">

```bash
curl -X POST http://127.0.0.1:3000/fetch_ptt_text \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <access-token>' \
  -d '{"message_id":""}'
```

</Tab>
<Tab label="Python">

```python
import requests

resp = requests.post(
    "http://127.0.0.1:3000/fetch_ptt_text",
    json={
        "message_id": ""
    },
    headers={"Authorization": "Bearer <access-token>"},  # 若已配置 access token
)
print(resp.json())
```

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"message_id":""}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/fetch_ptt_text", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
