<ActionHeader name={"get_rkey_server"} summary={"获取 rkey 服务器信息"} readOnly={true} category={"扩展"} />

通过 `POST http://127.0.0.1:3000/get_rkey_server`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

该动作无需参数。

## 返回 (data)

{ expired_time, name, private_rkey?, group_rkey? }：rkey 过期时间与（存在时的）私聊/群聊 rkey。

所有响应都包裹在标准信封中：`{ "status": "ok", "retcode": 0, "data": ... }`。下表描述 `data` 字段。

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `expired_time` | integer | ✓ | 过期时间戳（秒） |
| `name` | string | ✓ | 服务器名（恒 SnowLuma） |
| `private_rkey` | string | – | 私聊 rkey（存在时返回） |
| `group_rkey` | string | – | 群聊 rkey（存在时返回） |

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/get_rkey_server", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
