<ActionHeader name={"send_ark_share"} summary={"分享用户/群 Ark 卡片（NapCat 标准名）"} readOnly={true} category={"扩展"} />

通过 `POST http://127.0.0.1:3000/send_ark_share`（请求体为 JSON 参数）或 WebSocket 调用。

## 输入参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
| --- | --- | --- | --- | --- |
| `user_id` | uint | – |  |  |
| `group_id` | uint | – |  |  |
| `phone_number` | string | – | `""` |  |

## 返回 (data)

\{ arkMsg \}：服务端生成的推荐联系人 Ark 卡片 JSON 字符串。

所有响应都包裹在标准信封中：`{ "status": "ok", "retcode": 0, "data": ... }`。下表描述 `data` 字段。

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `arkMsg` | string | ✓ | Ark 卡片 JSON 字符串 |

<details>
<summary>原始 JSON Schema</summary>

```json
{
  "type": "object",
  "properties": {
    "arkMsg": {
      "type": "string",
      "description": "Ark 卡片 JSON 字符串"
    }
  },
  "required": [
    "arkMsg"
  ]
}
```

</details>

  -H 'Authorization: Bearer <access-token>' \

<Tab label="Go">

```go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
)

func main() {
	body := []byte(`{"user_id":10001,"group_id":123456,"phone_number":""}`)
	req, _ := http.NewRequest("POST", "http://127.0.0.1:3000/send_ark_share", bytes.NewReader(body))
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer <access-token>") // 若已配置 access token
	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```
